# Portfolio — Ignacio Barbosa Ferrer

Portfolio personal desarrollado con Astro. Diseño minimalista con arquitectura de componentes limpia.

🌐 **[Ver portfolio](https://IBarbosaF.github.io/portfolio-IBF)**

## Stack

- **Framework:** Astro 6
- **Estilos:** CSS nativo con variables y scoped styles
- **Tipografía:** DM Serif Display + DM Mono
- **Deploy:** GitHub Pages con CI/CD via GitHub Actions

## Estructura

```
src/
├── components/
│   ├── Navbar.astro
│   ├── HeroSection.astro
│   ├── SobreMi.astro
│   ├── Habilidades.astro
│   ├── Proyectos.astro
│   ├── Contacto.astro
│   └── Footer.astro
├── layouts/
│   └── Layout.astro
├── styles/
│   └── global.css
└── pages/
    └── index.astro
```
